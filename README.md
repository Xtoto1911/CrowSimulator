# Задача: Симуляция кормушки и ворон

Каждые **N секунд** прилетает новая ворона. Она действует по следующему алгоритму:

---

## 1. Попытка сразу сесть в кормушку
- Если в кормушке есть свободное место — ворона садится и начинает есть.

---

## 2. Если кормушка занята — садится на ветку
- Если на ветке есть свободное место — ворона ждёт.
- Если ветка полностью занята — ворона **сразу улетает**.

---

## 3. Ожидание места в кормушке
- Ворона может ждать **до 10 секунд**, пока в кормушке не освободится место.
- Если место освободилось — она **пересаживается в кормушку**, ест **M секунд**, затем улетает.
- Если место не освободилось за 10 секунд — ворона **улетает с ветки**.

---

## 4. Поедение корма
- Находясь в кормушке, ворона ест **M секунд**.
- По окончании еды ворона улетает.

---

## 5. Запрещено
- Выгонять ворон из кормушки нельзя.
- Перемещать ворон по принуждению нельзя.
